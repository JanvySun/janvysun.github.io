Maven 的安装与配置

发表于 2020-01-15 更新于 2021-06-30 分类于 Software 阅读次数：
本文字数： 1.4k 阅读时长 ≈ 3 分钟

Maven 是一个项目管理和综合工具。Maven 提供了开发人员构建一个完整的生命周期框架。开发团队可以自动完成项目的基础工具建设，Maven 使用标准的目录结构和默认构建生命周期。

安装

Maven 的安装非常方便，只需在 Maven 官网 上下载压缩包，解压即可完成安装。需要注意的是 Maven 的运行需要 JDK，Maven 对 JDK 有版本的要求，请尽量使用 JDK 1.8+ 的版本

注意下载的是 Binary 的压缩包

Windows 下 Maven 的安装

Windows 下直接将压缩包解压即可，解压之后如果要在命令行中使用 Maven，需要配置环境变量（环境变量中需要有 JAVA_HOME 的配置）。

右击「此电脑」，点击「高级系统设置」，点击「环境变量」加入 MAVEN_HOME（变量值为 Maven 的解压目录），然后在 PATH 变量中将 MAVEN_HOME 目录下的 bin 目录添加进去即可。
