IDEA 快捷键

发表于2020-03-12|更新于2020-03-12|Software
|阅读量:

功能（Main Menu）

  • 重写方法：Ctrl + Alt + O（Code -> Override Methods）
  • 实现方法：Ctrl + Alt + I（Code -> Implement Methods）
  • 代码生成：Alt + Shift + S（Code -> Generate…）
  • 变量引入：Alt + V（Refactor -> Extract -> Introduce Varialbe …）
  • 优化引入：Alt + I（Code -> Optimize import）
  • Back：Ctrl + Alt + Left、Ctrl + O（Navigate -> Back）
  • Forward：Ctrl + Alt + Right、Ctrl + I（Navigate -> Forward）
  • 代码提示：Alt + /（Code -> Completion -> Basic）
  • 智能提示：Alt + Shift + /（Code -> Completion -> Type Matching）
  • 代码包裹：Ctrl + Alt + T

编辑动作（Editor Actions）

  • 代码自动对齐：Ctrl + Alt + L
  • 字体放大：Ctrl + Wheel up（Increase Font Size）
  • 字体缩小：Ctrl + Wheel down（Decrease Font Size）
  • 字体恢复初始状态：Ctrl + Middle-Click（Reset Font Size）

工具窗（Tool Windows）

  • 项目目录（Project）：Alt + 1
  • 收藏夹（Favorites）：Alt + 2
  • 查找（Find）：Alt + 3
  • 启动（Run）：Alt + 4
  • 调试（Debug）：Alt + 5
  • 问题（Problems）：Alt + 6
  • 代码结构（Structure）：Alt + 7
  • 服务（Service）：Alt + 8
  • Git：Alt + 9
  • 版本控制工具（Commit）：Alt + 0
  • 终端（Terminal）：Alt + =（Terminal）
  • Hierarchy：Alt + H（Hierarchy，Ctrl + H 可将类加入其中进行分析）
  • Database：Alt + D（Database）
  • Maven：Alt + M（Maven）
