IDEA 的安装与配置

安装

Jetbrains 官网 下载安装程序，安装该过程不详细介绍，可以自定义安装路径。

注：本文使用的 IDEA 版本为 2021.1.3

插件

如果是联网环境，在 IDEA 内安装的插件均会被放到 AppData/Local/JetBrains/IntelliJIdea2021.1/plugins 目录下，可根据此路径进行备份。

推荐的插件有：IdeaVim、vuejs、Material Theme UI、Atom Material Icons


IDEA 的配置

字体配置

Settings -> Editor -> Font

推荐 Font 使用 JetBrains Mono，Fallback Font 使用 Microsoft Ya Hei

隐藏项目文件

Settings -> Editor -> File Types -> Ignored Files and Folders

由于项目的配置文件（如 .idea 目录、.iml 文件）是我们开发中用不到的，我们可以通过配置将其隐藏。在列表后追加 .idea;*.iml 即可。

代码模板

Settings -> Editor -> Live Templates

取消自动启动项目

Appearance & Behavior -> System Settings -> 取消勾选 Reopen last project on startup

取消双击 shift

Ctrl + Shift + A -> 输入 registry -> 勾选 ide.suppress.double.click.handler

取消光标闪烁

Editor -> General -> Appearance -> 取消勾选 Caret blinking

Java Import

Editor -> Code Style -> Java -> Imports

将其中的 Class count to use import with '*' 和 Names count to use static import with '*' 数值改成 99

SQL 配置

Languages & Frameworks -> SQL Dialects

将其中的 Sql Dialect 改为当前使用的类型，MyBatis xml 文件中的 SQL 语句即不会出问题

SQL Resolution Scopes 中的 Project mapping 也可以设置一下

代码补全

Editor -> General -> Code Completion -> 取消勾选 Match case

关闭共享索引

idea 为了能够更快的使用共享索引，会下载 JDK 和 Maven库的共享索引，给你的的项目代码构建共享索引。要关掉的也可以，进入 File –> Settings –> Tools 中，找到 Shared Indexes，改成： Ask before download， 或者 Don't download, use local indexes 然后再： File –> Invalidate Caches –> Clear downloaded shared indexes –> Invalidate and Restart

编码

Editor -> File Encodings

保证 Global Encoding 与 Project Encoding 为 UTF-8，Properties Files 为 UTF-8，Create UTF-8 files 为 with NO BOM

取消更新提醒

Appearance & Behavior -> System Settings -> Updates 将其中的检查更新关掉

关闭参数提示

Editor -> Inlay Hints，将其中的 Parameter names 关闭即可
