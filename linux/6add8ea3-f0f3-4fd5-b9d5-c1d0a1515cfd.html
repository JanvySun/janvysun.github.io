Linux 终端复用工具

screen

安装

Ubuntu: apt install screen

CentOS: yum install screen

注意，如果 screen 在非 root 用户下无法使用，则将 SCREENDIR 添加至环境变量：

使用

screen 命令格式：

常用：

选项：

  • -A：将所有的视窗都调整为当前终端机的大小。
  • -d<作业名称>：将指定的 screen 作业离线。
  • -h<行数>：指定视窗的缓冲区行数。
  • -m：即使当前已在作业中的 screen 作业，仍强制建立新的 screen 作业。
  • -r<作业名称>：恢复离线的screen作业。
  • -R：先试图恢复离线的作业。若找不到离线的作业，即建立新的 screen 作业。
  • -s<shell>：指定建立新视窗时，所要执行的 shell。
  • -S<作业名称>：指定 screen 作业的名称。
  • -v：显示版本信息。
  • -x：恢复之前离线的 screen 作业。
  • -ls或--list：显示当前所有的 screen 作业。
  • -wipe：检查当前所有的 screen 作业，并删除已经无法使用的 screen 作业。

在每个 screen session 下，所有命令都以 ctrl+a (C-a) 开始，操作如下：

  • C-a ?：Help，显示简单说明
  • C-a "：列出终端列表(列出详细信息)
  • C-a w：列出终端列表(在底部列出简单信息)
  • C-a A：重命名终端
  • C-a c：创建新的终端，并切换到该终端（注意这两个终端均在同一 screen session 下）
  • C-a d：暂时断开会话，但所有任务都依然执行(此时可以使用 screen -R 来恢复)
  • C-a k：关闭当前终端，并切换到下一个终端(当没终端时 退到原始shell终端)
  • C-a n：切换到下一个终端
  • C-a p：切换到上一个终端
  • C-a t：显示当前时间，和系统的 load
  • C-a S：新建水平线分割终端
  • C-a Tab：水平线切换终端
  • C-a X：关闭当前终端
  • C-a +：扩大当前终端
  • C-a -：缩小当前终端
  • C-a [Space]：窗口切换
  • C-a 0-9：在 0-9 个终端之间切换
  • C-a C-a：最近使用的 2 个终端切换
  • C-a x：锁住当前窗口，需用用户密码解锁
  • C-a z：把当前会话放到后台执行，用 shell 的 fg 命令则可回去。
  • Ctrl-a <Esc>：进入选择模式，此时按 Space 开始选择，光标移动方式与 vim 一致，再按 Space 结束选择，此时所选内容将被复制到缓冲区
  • Ctrl-a ]：粘贴缓冲区的内容
  • C-a :：进入末行模式，可执行 screen 内的命令
    • screen <command>：新建 screen 终端，并运行命令 command
    • resize <height>：改变当前窗口的高度为 height
    • quit：退出 screen 并关闭所有终端，结束其中的所有任务

tmux

安装

Ubuntu：apt install tmux

CentOS：yum install tmux

使用

常用：

与 Screen 不同的是，tmux 中的控制命令以 Ctrl+b(C-b) 开始

会话操作：

  • C-b ?：列出所有快捷键；按 q 返回
  • C-b d：脱离当前会话；这样可以暂时返回Shell界面，输入tmux attach能够重新进入之前的会话
  • C-b D：选择要脱离的会话；在同时开启了多个会话时使用
  • C-b Ctrl+z：挂起当前会话
  • C-b r：强制重绘未脱离的会话
  • C-b s：选择并切换会话；在同时开启了多个会话时使用
  • C-b :：进入命令行模式；此时可以输入支持的命令，例如 kill-server 可以关闭服务器
  • C-b [：进入复制模式；此时的操作与 vi/emacs 相同，按 q/Esc 退出
  • C-b ~：列出提示信息缓存；其中包含了之前 tmux 返回的各种提示信息

窗口操作：

  • C-b c：创建新窗口
  • C-b &：关闭当前窗口
  • C-b 数字键：切换至指定窗口
  • C-b p：切换至上一窗口
  • C-b n：切换至下一窗口
  • C-b l：在前后两个窗口间互相切换
  • C-b w：通过窗口列表切换窗口
  • C-b ,：重命名当前窗口；这样便于识别
  • C-b .：修改当前窗口编号；相当于窗口重新排序
  • C-b f：在所有窗口中查找指定文本
  • C-b PgUp：进入历史查看模式，使用方向键或 PgUp，PgDown 来滚动

面板操作：

  • C-b "：将当前面板水平分割
  • C-b %：将当前面板垂直分割
  • C-b x：关闭当前面板
  • C-b !：将当前面板置于新窗口；即新建一个窗口，其中仅包含当前面板
  • C-b z：当前窗格全屏显示，再使用一次会变回原来大小
  • C-b 方向键：移动光标以选择面板
  • C-b Ctrl+方向键：以1个单元格为单位移动边缘以调整当前面板大小
  • C-b Space：在预置的面板布局中循环切换；依次包括even-horizontal、even-vertical、main-horizontal、main-vertical、tiled
  • C-b q：显示面板编号
  • C-b o：在当前窗口中选择下一面板
  • C-b {：向前置换当前面板
  • C-b }：向后置换当前面板
  • C-b Alt+o：逆时针旋转当前窗口的面板
  • C-b Ctrl+o：顺时针旋转当前窗口的面板
