Linux 常用命令

发表于 2020-05-13 更新于 2021-07-10 分类于 Linux 阅读次数：
本文字数： 18k 阅读时长 ≈ 37 分钟

Linux 是一种自由和开放源码的类 UNIX 操作系统，使用 Linux 内核。目前存在着许多不同的 Linux 发行版，可安装在各种各样的电脑硬件设备。各版本的 Linux 使用方式大致相同，本文介绍 Linux 下的常用命令。

Linux 的命令就是提醒 shell 程序找到键入命令所对应的可执行程序或代码，并由其分析后提交给内核分配资源将其运行起来，表现为一个或多个进程

使用方法如下：

其中选项可分为短选项和长选项，用于增加命令功能

  • 短选项：-char ，一般是用一个字符表示，多个短选项可以组合
  • 长选项：–word ，用完整单词或单词缩写来表示

参数：命令作用的对象（注：有些选项是可以带参数的）

获得命令的使用帮助：

  • 内部命令：help + 命令
  • 外部命令：命令 + –help

基本命令

pwd

pwd（Printing Working Directory），输出工作目录（当前目录），用的是 绝对路径 表示

ls

ls（list）：显示当前目录下的所有文件，命令格式如下：

各选项的作用：

  • -l：以长格式显示文件信息，文件信息具体内容如下：
    首先是文件类型：
    -：普通文件（f）
    d：目录文件
    b：块设备文件
    c：字符设备文件
    l：符号链接文件（软链接）
    p：命令管道文件
    s：套接字文件
    然后的9位字符是文件权限，r（读）、w（写）、x（执行），每3位一组，第一组表示文件属主对其的权限，第二组表示文件属组中的成员对其的权限，第三组表示其他成员对其的权限
    后一位数字表示文件硬链接次数
    然后是文件属主（owner）和文件属组（group）
    之后是文件大小和改变时间
  • -h：单位转换，将字节单位转换为K或者M等，一般与-l配合使用
  • -a：显示包括隐藏文件在内的所有文件，其中，“.”表示当前目录，“..”表示上级目录
  • -A：显示所有文件，不包括.和..
  • -d：显示目录自身，一般与-l配合使用
  • -i：显示目录下每个文件的索引结点号（index node）
  • -r：逆序显示文件（默认升序）
  • -R：递归显示文件

参数：ls 后可跟参数，默认参数为 当前工作目录

cd

cd（change directory）：切换工作目录

  • cd 后不跟参数是回到当前用户的 home 目录
  • cd ~[USERNAME]：进入指定用户的 home 目录，默认为当前用户
  • cd -：在最近使用的 2 个目录之间来回切换

注：cd 后的参数只能是目录文件

type

显示命令属于哪种类型（命令类型：内置命令和外部命令）

内置命令由 shell 内建，外部命令是在文件系统中的某个路径下有一个与命令名相同的可执行文件

type 后面的参数跟的是某个命令

which

which COMMAND：此命令可显示命令所在路径

date

date 命令用于显示当前时间（系统时间）

-s 选项用于设置日期和时间

  • date +%F 可按「年-月-日」显示时间
  • date +%F-%H-%M-%S 可按「年-月-日-时-分-秒」显示时间

man

man COMMAND：显示命令手册

注意：man 是分章节的，可用 whatis COMMAND 来查看该命令出现在哪些章节

man 后可跟章节名，如：man 2 read

章节：

  1. 可执行程序或者 shell 命令（/bin, /usr/bin, /usr/local/bin）
  2. 系统调用（内核提供的函数)
  3. 库调用（程序库中的函数）
  4. 特殊文件（设备文件,通常位于/dev）
  5. 文件格式和规范（配置文件语法,如/etc/passwd）
  6. 游戏
  7. 杂项（包括宏包和规范）
  8. 系统管理命令（/sbin, /usr/sbin, /usr/local/sbin）

命令手册（man）中

  • NANE: 命令名称及功能简要说明
  • SYNOPSIS: 用法说明，包括可用选项
  • DESCRIPTION: 功能详细说明
  • OPTIONS: 各选项含义
  • FILES: 相关配置文件
  • EXAMPLES: 使用示例

在命令手册中翻屏：

  • 向后翻一屏：SPACE（空格键）
  • 向前翻一屏：b
  • 向后翻一行：ENTER（回车键）
  • 向前翻一行：k

按 q 键退出命令手册

grep

该命令用于搜索文件中匹配模式的行，格式如下：

其中模式(PATTERN)可以是字符串，也可以是正则表达式

选项如下：

  • -i：忽略字符大小写
  • --color：将搜索到的字符加颜色
  • -v：反向查找，被模式匹配到的行不显示，只显示没有匹配到的行
  • -o：只显示被模式匹配到的字符串（每个串显示为一行）
  • -E：使用扩展正则表达式
  • -A NUM：表示匹配到某一行之后，此行的下NUM行也显示出来
  • -B NUM：表示匹配到某一行后，此行的前面NUM行也显示出来
  • -C NUM：表示匹配到某一行后，此行的前后各NUM行也显示出来

正则表达式（贪婪模式，尽可能长的匹配）：

  • .：匹配任意单个字符
  • []:匹配指定范围内的任意单个字符（对 [] 字符集同样有效）
  • [^]:匹配指定范围外的任意单个字符
  • *：匹配其前面的字符任意次
  • .*：任意长度的任意字符
  • ?：匹配其前面的字符1或0次
  • \{m,n\}:匹配其前面的字符至少 m 次，至多 n 次
    如：\{1,\} 最少一次，多了不限
  • ^：其后的所有内容必须出现在行首
  • $：此字符的前面的所有内容必须出现在行尾
  • ^$：空白行
  • \<(或者\b):其后面的所有字符必须作为单词首部出现
  • \>(或者\b):其前面的所有字符必须作为单词尾部出现
    这两个可以结合使用：如<root>` 就是匹配单个 root 单词（此单词只能为 root，注意单词前后不能有其他字母）
  • \(\):分组，\ 为转义字符，将 () 转为它本来的意义。如：\(ab\)* 就是匹配含有任意个连续 ab 的行。关于分组还有一个后向引用，在分组 () 后用 \1 表示第一个括号匹配的内容， \2 表示第二个括号匹配的内容，\3 表示第三个括号匹配的内容，以此类推。

扩展正则表达式：
相对于正则表达式增加了 +，匹配其前面的字符至少 1 次；同时花括号 {} 不需要用转义字符 \；分组 () 也不需要使用转义字符 \；增加了或运算符 |，匹配其左边的所有字符或者其右边的所有字符，通常和分组配合使用

grep -E 和 egrep 的效果是一样的，都是使用扩展正则表达式。

touch

该命令用于修改文件的时间戳（3 个时间都改），常用此命令创建文件

选项：

  • -c：不创建新文件，改变3个时间
  • -a：仅改变文件访问（access）时间
  • -m：仅改变文件更改时间（modification）
  • -t：使用过去的时间，格式为 年月日时分.秒

文体时间如下：

  • access time：访问访问时间，读取文件内容就会改变
  • modify time：修改时间，该变文件内容的时候该时间就会改变
  • change time：改变时间，文件元数据发生改变的时候该时间就会改变

注：每一次使用 touch ，最近改动时间均会改变

find

find 命令用于在某个路径下对某文件名进行查找，格式如下：

查找方式是实时查找，能够精确匹配文件名，支持众多查找标准。通过遍历指定目录中的所有文件完成查找，查找速度较慢

find 命令格式：

其中：

  • 查找路径：默认为当前路径
  • 查找标准：默认为指定路径下的所有文件
  • 处理操作：默认为显示（输出到屏幕）

find 的匹配标准：

  • -name 'FILENAME'：对文件名进行精确匹配
    FILENAME 中可以使用文件名通配，如 *（任意长度任意字符）、?、[] 等
  • -iname 'FILENAME'：文件名匹配时忽略大小写
  • -regex PATTERN：基于正则表达式进行文件名匹配
  • -user USERNAME: 根据属主查找
  • -group GROUPNAME: 根据属组查找
  • -uid UID: 根据UID查找
  • -gid GID: 根据GID查找
  • -nouser：查找没有属主的文件
  • -nogroup: 查找没有属组的文件
  • -type TYPENAME：根据文件类型查找
    f：普通文件
    d：目录文件
    b：块设备文件
    c：字符设备文件
    l：符号链接文件（软链接）
    p：命令管道文件
    s：套接字文件
  • -size [+|-]SIZE：根据文件大小来查找
    如果SIZE前不跟加号（+）或者减号（-）就表示精确匹配文件大小（注意是近似精确值），如果跟加号就表示匹配大于 SIZE 的文件，跟减号就表示是匹配小于 SIZE 的文件，SIZE 后可跟单位（k、M、G），默认为字节

根据时间匹配：

  • -mtime [+|-]TIME：根据更改时间进行匹配
  • -atime [+|-]TIME：根据访问时间进行匹配
  • -ctime [+|-]TIME：根据修改时间进行匹配

注意：以上是最近根据天数进行匹配
-mmin、-amin、-cmin 是根据最近分钟数进行匹配

根据权限匹配：

  • -perm MODE：精确根据权限匹配
  • /MODE：只要 3 位中任何一位匹配就能匹配到
  • -MODE：文件权限能够完全包含此 MODE 即可匹配，如 755(rwxr-xr-x) 完全包含 644(rw-r–r–)

组合条件：

  • -a：与
  • -o：或
  • -not：非

如要在当前目录下查找文件大小大于 1M 并且属主是 root 的文件：find ./ -size +1M -a -user root

找到后的操作

  • -print：显示
  • -ls：类似ls -l显示找到后的每个文件

  • sy：内核空间占据CPU比例
  • id：空闲比例
  • wa：等待io完成所消耗的时间比例
  • st：被虚拟化”偷走”的时间比例

pmap

  • -x：显示详细格式的信息

kill

此命令用于杀死某个进程，也可用于进程间的信号通信

使用 kill -l 可以查看所有的信号，其中常用的信号有：1 SIGHUP（让一个进程不用重启就可以读取配置文件，并让新的配置文件生效）；2 SIGINT（此信号相当于Ctrl+c，中断一个进程）；9 SIGKILL（杀死一个进程）；15 SIGTERM（终止一个进程，此信号为终止进程的默认信号）

kill 用法：kill [-signal] PID
其中 -signal 可以指定信号号码，如-1即指定SIGHUP，也可指定信号名称，如-SIGHUP，还可以指定名称简写，如-KILL即指定SIGKILL

如果要结束一个命令的所有进程，还可以使用 killall COMMAND

kill 命令还可以终止某作业：kill %JOBID

exec

启动一个进程，并让这个进程直接替换父进程来执行，而不是以父进程的子进程执行

trap

此命令用于捕捉信号

就是当看见信号的时候执行COMMAND命令然后就捕捉掉这个信号
注意：9号信号和15号信号不能捕捉；COMMAND命令中可以有多条命令，用分号隔开。

同样的，COMMAND命令也可以是一个函数，这样就可以将许多命令放在一个函数中执行了

at

此命令用于在未来的某个时间执行某个作业
用法 at TIME，然后会出现>提示符，提示输入命令，输完命令后按Ctrl+d键结束
其中，时间可用绝对时间，如HH:MM, DD.MM.YY, MM/DD/YY
也可使用相对时间：now+#，单位有minutes，hours，days，weeks

  • -l：显示将运行的作业
  • -d JOBID：删除某个作业

作业执行完毕后是以邮件发给用户的

batch

此命令和 at 命令类似，也是在某个时间执行某个任务，和at不同的是，batch不能指定时间，而是让系统自行选择空闲时间去执行某个指定的任务

netstat

此命令用于显示网络相关状态

  • 显示网络连接

    • -t：tcp协议相关
    • -u：udp协议相关
    • -w：raw socket相关
    • -l：处于监听状态
    • -a：所有状态
    • -n：以数字显示IP和端口
    • -e：扩展格式
    • -p：显示相关进程及PID

    常用组合：
    -tan, -uan, -tnl, -unl

  • 显示路由表

    • -r：显示内核路由表
    • -n：以数字显示

  • 显示接口统计数据

ip

此命令用于配置网络

OBJECT：={ link | addr | route }

  • link：网络接口属性相关
    • show：查看接口属性，如 ip -s link show
    • set：配置接口，如 ip link set DEV up|down
  • addr：协议地址相关
    • add：添加地址，如 ip addr add ADDRESS dev DEV，ip addr add 192.168.0.1/24 dev eth0，此命令中还可以通过 label 指定别名
    • del：删除地址，ip addr del ADDRESS dev DEV
    • show：显示信息，如：ip addr show dev DEV to PREFIX
    • flush：清空，如：ip addr flush dev DEV to PREFIX
  • route：路由相关

含有各种子命令，通过man ip查看，不一一列举。如：ip route add to 10.0.1.0/24 dev eth1 via 192.168.100.6。常用子命令有：add, change, show, flush, replace。

ss

网络状态查看工具

选项：

  • -t：tcp协议相关
  • -u：udp协议相关
  • -w：裸套接字相关
  • -x：unix sock相关
  • -l：listen状态的连接
  • -a：所有
  • -n：数字格式
  • -p：相关的程序及PID
  • -e：扩展的信息
  • -m：内存用量
  • -o：计时器信息

FILTER：=[ state TCP-STATE ] [ EXPRESSION ]

TCP常见的状态：

  • LISTEN：监听
  • ESTABLISHED：已建立的连接
  • CLOSED：关闭

常用组合：-tan, -tanl, -tanlp, -uan, -tunlp
