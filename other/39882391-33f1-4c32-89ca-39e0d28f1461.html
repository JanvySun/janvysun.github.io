Markdown 语法

发表于 2020-02-14 更新于 2021-06-30 分类于 Other 阅读次数：
本文字数： 2.4k 阅读时长 ≈ 5 分钟

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的 XHTML（或者HTML）文档。本文介绍 Markdown 语法。

标题

Markdown 标题支持两种形式的语法：类 Setext 和 类 Atx 形式

类 Setext

类 Setext 形式是使用底线的形式，利用 = (最高阶标题) 和 - (第二阶标题)，例如：

效果与 HTML 下 一级标题（h1 标签） 与 二级标题（h2 标签） 相同

类 Atx

类 Atx 形式是使用 # 来设置标题。一个 # 为一级标题，两个 # 为二级标题，依此类推，最多 6 个 # 用于设置六级标题。

可以选择性地「闭合」类 atx 样式的标题，这纯粹只是美观用的。若是觉得这样看起来比较舒适，你就可以在行尾加上 #，而行尾的 # 数量也不用和开头一样。
