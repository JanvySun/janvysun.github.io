Selenium 的安装与使用

发表于 2021-03-30 更新于 2021-06-30 分类于 Python 阅读次数：
本文字数： 2.3k 阅读时长 ≈ 5 分钟

Selenium 是一个 Web 的自动化测试工具，最初是为网站自动化测试而开发的。Selenium 可以直接运行在浏览器上，它支持所有主流的浏览器，可以接收指令，让浏览器自动加载页面，获取需要的数据，甚至页面截屏。使用 Selenium 可以让我们加载动态的页面，也就是说我们使用 Selenium 获取到的内容就是浏览器加载完毕的内容。本文介绍 Selenium 的安装与基础使用。

常用方法

页面元素定位

  • find_element_by_id
    该方法会根据id来查找元素，返回一个元素
  • find_element_by_class_name
    该方法通过元素类名来获取元素，注意该方法只能传入一个class_name
    与之对应的是find_elements_by_class_name方法，返回一个列表
  • find_element_by_xpath
    该方法可以通过xpath语法来查找元素，返回第一个找到的元素
    该方法传入的参数就是xpath语法，但是需要注意的是:
    该方法找到的只能是一个元素而不能是文本或者属性
    如果想要获取元素的属性或文本需要再调用元素的get_attribute()/text方法
  • find_elements_by_xpath
    该方法通过xpath语法来查找元素，返回一个列表，里面包含所有匹配的元素
    如下：
  • find_element_by_link_text
    该方法通过元素的文本内容来获取元素，传入的参数需和元素的文本内容一致才能匹配
    与之对应的为find_elements_by_link_text方法，返回的是一个列表
  • find_element_by_partial_link_text
    该方法和上面的方法一样，也是通过元素的文本内容来获取元素，但是它只需要参数和元素内容部分匹配即可
    与之对应的为find_elements_by_partial_link_text，返回的是一个列表

selenium 注意事项

获取文本和获取属性

  • 先定位到元素，然后调用.text或者get_attribute方法来获取元素文本或属性
  • selenium获取的页面数据是浏览器中elements的内容
  • find_element和find_elements的区别
    • find_element返回一个element，如果没有会报错
    • find_elements返回一个列表，没有就是空列表
    • 在判断是否有下一页的时候，使用find_elements来根据结果的列表长度来判断

内嵌 frame

如果页面中含有 iframe、frame，需要先调用 driver.switch_to.frame 的方法切换到 frame 中才能定位元素

翻页

selenium 请求第一页的时候会等待页面加载完了之后再获取数据，但是在点击翻页之后，会直接获取数据，此时可能会报错，因为数据还没有加载出来，需要等待几秒
