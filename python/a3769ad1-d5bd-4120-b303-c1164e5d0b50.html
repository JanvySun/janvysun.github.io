可变类型参数与不可变类型参数

在 Python 中，可变类型只有 2 种，就是列表和字典，其他的变量类型都是不可变类型

Python 中函数参数是引用传递（和 Java 类似，相当于引用和变量本身指向同一块内存区域）

对于不可变类型，因变量名不能修改，所以运算的时候不会影响到变量自身

而对于可变类型来说，函数体中的运算有可能会更改传入的参数变量

不定长参数

有时可能需要一个函数能处理比当初声明时更多的参数

这些参数叫做不定长参数，声明时不会命名。

加了星号 * 的变量 args 会存放所有未命名的变量参数，args 为元组

而加两个星号 ** 的变量 kwargs 会存放命名的变量参数

即形如 key = value 的参数，kwargs 为字典

其中的 args 本身是将变量保存为元组的，因此也可以直接传元组进来，不过要在元组名前加上 * 号进行传递

kwargs 是将变量保存为字典的，也可以将字典直接传进来，在字典名上加 2 个 * 号进行传递即可
