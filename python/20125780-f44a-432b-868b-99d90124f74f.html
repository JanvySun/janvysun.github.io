Python 列表、元组与集合

列表

Python 中的列表和 Java 语言中的 list 类似，但是其比 list 强大的地方在于列表中的元素可以是不同类型的

如：list = [1, "abcd", 1.2, False]

要想得到列表中的某个元素，可以使用下标来进行操作

如 list[0] 代表第一个元素 1，list[1] 代表第二个元素”abcd”，依此类推

元组

元组和列表类似，不同之处在于元组中的元素不能修改，元组使用小括号，列表使用方括号

如：aTuple = ("et", 11, 99.9)

元组的访问和列表一样，也是通过下标对其进行访问，但是不能修改其值，元组也不能删除其中的某个值

元组中有的内置函数 count, index 用法和列表一样，分别用来计数和定位

集合

集合类似于元组，不过其中的元素可以添加，但是其中的元素不能有重复。

集合是用大括号描述其元素的，如： a = {1,2,3,4}
