Python 字符串

发表于2021-04-28|更新于2021-04-28|Python
|字数总计:1.6k|阅读时长:5分钟|阅读量:

字符串

  • 在 Python 中，双引号或者单引号中的数据就是字符串
  • 字符串的格式化输出可以用 %s 来进行输出
  • 另外，input 函数接收的值就是字符串
  • 还可以用 str() 函数将其他类型的变量转换为字符串类型的变量



字符串下标

字符串的下标和 C 语言的一样，第一个字符下标为 0

不同的是 STR[-1] 表示字符串的倒数第一个字符，STR[-2] 表示字符串倒数第二个字符，以此类推



字符串截取

Python 还支持字符串截取，是指对操作的字符串截取其中一部分的操作。

在 Python 中，字符串、列表、元组 都支持截取操作。

字符串截取语法：STR[起始位置:结束位置:步长]

注意：

  • 截取不包括结束位置的那个字符
  • 还有，两个位置如果省略的话，Python解释器会自动根据步长来查找最终位置，步长省略默认为1

因此可以用 STR[::-1] 来进行字符串的翻转，当然也可以写成 STR[-1::-1]



转义字符

在需要在字符中使用特殊字符时，python 用反斜杠 \ 转义字符。如下表：

转义字符描述
\在行尾时表示续行符
\\反斜杠符号
\'单引号
\"双引号
\a响铃
\b退格(Backspace)
\e转义
\000空
\n换行
\v纵向制表符
\t横向制表符
\r回车
\f换页
\oyy八进制数，y 代表 0~7 的字符，例如：\012 代表换行。
\xyy十六进制数，以 \x 开头，yy 代表的字符，例如：\x0a代表换行
\other其它的字符以普通格式输出



字符串格式化

Python 支持格式化字符串的输出。尽管这样可能会用到非常复杂的表达式，但最基本的用法是将一个值插入到一个有字符串格式符 %s 的字符串中。

在 Python 中，字符串格式化使用与 C 中 sprintf 函数一样的语法。

python 字符串格式化符号:

符号描述
%c格式化字符及其 ASCII 码
%s格式化字符串
%d格式化整数
%u格式化无符号整型
%o格式化无符号八进制数
%x格式化无符号十六进制数
%X格式化无符号十六进制数（大写）
%f格式化浮点数字，可指定小数点后的精度
%e用科学计数法格式化浮点数
%E作用同 %e，用科学计数法格式化浮点数
%g%f 和 %e 的简写
%G%F 和 %E 的简写
%p用十六进制数格式化变量的地址

格式化操作符辅助指令:

符号功能
*定义宽度或者小数点精度
-用做左对齐
+在正数前面显示加号( + )
<sp>在正数前面显示空格
#在八进制数前面显示零(‘0’)，在十六进制前面显示’0x’或者’0X’(取决于用的是’x’还是’X’)
0显示的数字前面填充’0’而不是默认的空格
%%% 输出一个单一的 %
(var)映射变量(字典参数)
m.n.m 是显示的最小总宽度，n 是小数点后的位数(如果可用的话)

Python 2.6 开始，新增了一种格式化字符串的函数 str.format()，它增强了字符串格式化的功能。

常用函数

find/rfind

find 函数用于检测某个子串是否包含在字符串中，如果是就返回第一次找到的开始的索引值，否则返回 -1

find 函数中的 start 和 end 分别是字符串的其实和结束位置，可以指定从某个位置开始查找

find 是从左往右找，而 rfind 是从右往左找

index/rindex

用法和 find 函数一样，只不过如果没找到就会报错

index 是从左往右找，而 rindex 是从右往左找

count

此函数返回子串 str 在 STRING 中的 start 和 end 之间里出现的次数

replace

此函数将 STRING 中的 str1 替换成s tr2，如果 count 指定，则替换不超过 count 次

注意：该函数不能直接修改字符串，而是将修改过的值返回出来，字符串本身是没有改变的

split/rsplit

以 str 为分隔符分开 STRING，如果指定 maxsplit 的值则仅分隔 maxsplit 个字符串，返回一个列表

split 中的 sep 默认为空格，maxsplit 默认为 -1，就是切完所有的

split 是从左往右，rsplit 是从右往左

capitalize

把字符串的第一个字母大写

title

把字符串的每个单词首字母大写

startswith/endswith

判断字符串 STRING 是否以 str 开头/结尾，是则返回 True，否则返回 False

lower/upper

将 STRING 中的所有字母改为大写/小写

ljust/rjust/center

将一个字符串 STRING 左对齐/右对齐/居中，并使用空格将字符串填充至 width 长度

lstrip/rstrip/strip

删除字符串 STRING 左边/右边/两边的空白字符

partition/rpartition

将 STRING 以 str 分割成 3 个部分，返回元组

r 是从右往左找到第一个 str

splitlines

按照行分隔，返回一个包含各行作为元素的列表

isalpha/isdigit/isalnum/isspace

如果字符串 STRING 只包含字母/数字/字母和数字/空格，则返回 True，否则返回 False

join

将某个列表 list 中的元素以 STRING 为分隔符组成一个新的字符串
