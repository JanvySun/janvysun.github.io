Python 基础

发表于 2019-04-13 更新于 2021-06-30 分类于 Python 阅读次数：
本文字数： 43k 阅读时长 ≈ 1:30

Python是一种跨平台的计算机程序设计语言。 是一个高层次的结合了解释性、编译性、互动性和面向对象的脚本语言。最初被设计用于编写自动化脚本(shell)，随着版本的不断更新和语言新功能的添加，越多被用于独立的、大型项目的开发。本文介绍 Python 的基础内容。

解释器

计算机不能直接理解任何除机器语言以外的语言，所以必须要把程序员所写的程序语言翻译成机器语言，计算机才能执行程序。
将其他语言翻译成机器语言的工具，被称为编译器。
编译器翻译的方式有两种：一个是编译，另外一个是解释。
两种方式之间的区别在于翻译时间点的不同。当编译器以解释方式运行的时候，也称之为解释器。

编译型和解释型语言工作对比：

  • 编译型语言
    程序在执行之前需要一个专门的编译过程。把程序编译成为机器语言的文件，运行时不需要重新翻译，直接使用编译的结果就行了。程序执行效率高，依赖编译器，跨平台性差些。如 C、C++ 等。

  • 解释型语言
    解释型语言编写的程序不进行预先编译，以文本方式存储程序代码，会将代码一句一句直接运行。在发布程序时，看起来省了道编译工序，但是在运行程序的时候，必须先解释再运行。

Python 程序

Python 程序的文件扩展名通常都是 .py

交互式 Python

交互式 Python 是直接在终端中运行解释器，而不输入要执行的文件名。

在 Linux 中直接输入 python 命令即可进入 Python 的交互式界面，然后输入 Python 的代码，会立即看到程序执行结果。在交互式界面中输入 exit() 即可退出，或者按 Ctrl+D 也能退出。

程序用来操作数据，而变量是用来存储数据

等号 = 用来给变量赋值，等号左边是一个变量名，等号右边是存储在变量中的值。

因此定义变量的方法：变量名 = 值

变量类型

在内存中创建一个变量，会包括：
1.变量的名称
2.变量保存的数据
3.变量存储数据的类型
4.变量的地址

在Python中，定义变量时是不需要指定变量的类型的
Python解释器运行的时候，会自动根据赋值语句等号右侧的数据来自动推导出变量中保存数据的准确类型
变量类型分为数字型和非数字型
其中数字型有: 整型(int)、浮点型(float)、布尔型(bool)
非数字型有: 字符串(string)、列表、元组、字典等

如果想要查看某个变量的类型，在交互式界面下可以使用 type(NAME) 函数来查看
在Python中，数字型变量之间是可以直接计算的
如果变量是bool类型的在计算时True对应的值为1，False对应的值为0
字符串的计算：字符串之间可以使用”+”号拼接生成新的字符串，也可以使用”*”号来重复拼接相同的字符串。
数字型变量和字符串之间不能进行其他计算

所谓输入，就是用代码获取用户通过键盘输入的信息
要通过变量读取键盘上的信息，需要用到 input 函数
用法：input("提示信息")
该函数的返回值就是从键盘读取到的内容，可以用某个变量来接收该值，如:
VALUE = input()
用户输入的任何内容该函数都会返回字符串（Python3中）
Python 2 和Python 3的input是有区别的：
Python2输入的时候会将输入的结果当做命令来看待
而Python3的输入结果就是字符串，Python3中的input就相当于Python2中的raw_input

字符串

在Python中，双引号或者单引号中的数据就是字符串
字符串的格式化输出可以用%s来进行输出
另外，input函数接收的值就是字符串
还可以用str()函数将其他类型的变量转换为字符串类型的变量

字符串的下标和C语言的一样，第一个字符下标为0
不同的是STR[-1]表示字符串的倒数第一个字符，STR[-2]表示字符串倒数第二个字符，以此类推

Python还支持字符串切片，所谓的切片就是指对操作的对象截取其中一部分的操作。
字符串、列表、元组都支持切片操作
字符串切片语法：STR[起始位置:结束位置:步长]
切片不包括结束位置的那个字符
还有，两个位置如果省略的话，Python解释器会自动根据步长来查找最终位置，步长省略默认为1
因此可以用STR[::-1]来进行字符串的翻转，当然也可以写成STR[-1::-1]

字符串常用方法：

  • find/rfind
    find函数用于检测某个子串是否包含在字符串中，如果是就返回第一次找到的开始的索引值，否则返回-1
    STRING.find(str[, start, end])
    find函数中的start和end分别是字符串的其实和结束位置，可以指定从某个位置开始查找
    find是从左往右找，而rfind是从右往左找
  • index/rindex
    用法和find函数一样，只不过如果没找到就会报错
    STRING.index(str[, start, end])
    index是从左往右找，而rindex是从右往左找
  • count
    此函数返回子串str在STRING中的start和end之间里出现的次数
    STRING.count(str, start, end)
  • replace
    此函数将STRING中的str1替换成str2，如果count指定，则替换不超过count次
    STRING.replace(str1, str2[, STRING.count(str1)])
    注意：该函数不能直接修改字符串，而是将修改过的值返回出来，字符串本身是没有改变的
  • split/rsplit
    以str为分隔符切片STRING，如果指定maxsplit的值则仅分隔maxsplit个字符串，返回一个列表
    STRING.split([sep=str, maxsplit=NUM])
    split中的sep默认为空格，maxsplit默认为-1，就是切完所有的
    split是从左往右，rsplit是从右往左
  • capitalize
    把字符串的第一个字母大写
    STRING.capitalize()
  • title
    把字符串的每个单词首字母大写
  • startswith/endswith
    判断字符串STRING是否以str开头/结尾，是则返回True，否则返回False
    STRING.startwith(str)
    STRING.endswith(str)
  • lower/upper
    将STRING中的所有字母改为大写/小写
    STRING.lower()
    STRING.upper
  • ljust/rjust/center
    将一个字符串STRING左对齐/右对齐/居中，并使用空格将字符串填充至width长度
    STRING.ljust/rjust/center(width)
  • lstrip/rstrip/strip
    删除字符串STRING左边/右边/两边的空白字符
    STRING.lstrip/rstrip/strip()
  • partition/rpartition
    将STRING以str分割成3个部分，返回元组
    STRING.[r]partition(str)
    r是从右往左找到第一个str
  • splitlines
    按照行分隔，返回一个包含各行作为元素的列表
  • isalpha/isdigit/isalnum/isspace
    如果字符串STRING只包含字母/数字/字母和数字/空格，则返回True，否则返回False
    STRING.isalpha/digit/alnum/space()
  • join

元组

元组和列表类似，不同之处在于元组中的元素不能修改，元组使用小括号，列表使用方括号
如：aTuple = ("et", 11, 99.9)
元组的访问和列表一样，也是通过下标对其进行访问，但是不能修改其值，元组也不能删除其中的某个值
元组中有的内置函数count, index用法和列表一样，分别用来计数和定位

字典

字典和列表类似，也能够存储多个数据。
和列表不同的是，字典存放数据是按照key:value的方式存储的
列表中找某个元素时是根据下标来进行的，而字典找某个元素时是根据键(key)来查找值(value)的
字典中的每个元素都由2部分组成，key:value
例如：info就是一种字典
info = {'name':'班长', 'id':100, 'sex':'f', 'address':'北京'}
可以根据键访问值：
print(info['name'])
print(info['id'])

  • r
    以只读方式打开文件。
    文件的指针将会放在文件的开头。这是默认模式，如果只有r的话可以不写
  • w
    打开一个文件只用于写入。
    如果该文件已存在则将其覆盖。如果该文件不存在，创建新文件。
  • a
    打开一个文件用于追加。

    以二进制格式打开一个文件用于只读。
    文件指针将会放在文件的开头。
  • wb
    以二进制格式打开一个文件只用于写入。
    如果该文件已存在则将其覆盖。如果该文件不存在，创建新文件。
  • ab
    以二进制格式打开一个文件用于追加。
    如果该文件已存在，文件指针将会放在文件的结尾。也就是说，新的内容将会被写入到已有内容之后。
    如果该文件不存在，创建新文件进行写入。
  • r+
    打开一个文件用于读写。文件指针将会放在文件的开头。
  • w+
    打开一个文件用于读写。
    如果该文件已存在则将其覆盖。
    如果该文件不存在，创建新文件。
  • a+
    打开一个文件用于读写。
    如果该文件已存在，文件指针将会放在文件的结尾。
    文件打开时会是追加模式。如果该文件不存在，创建新文件用于读写。

面向对象

在编写程序的时候我们往往用的是面向过程的编程思维，面向过程就是根据业务逻辑从上到下写代码。
这种编程思维最容易被初学者接收，但是有一定的缺陷，就是代码可维护性很差，要修改某一个功能的时候往往需要修改很多内容。因此就需要有一种其他的编程思维来改变。
面向对象就可以解决这种问题，其思维是将数据与函数绑定到一起，进行封装，这样能够更快速的开发程序，减少代码的重写过程。
面向对象(object-oriented;简称:OO)至今还没有统一的概念 我这里把它定义为：按人们认识客观世界的系统思维方式,采用基于对象(实体)的概念建立模型,模拟客观世界分析、设计、实现软件的办法。
面向对象编程(Object Oriented Programming-OOP)是一种解决软件复用的设计和编程方法。
这种方法把软件系统中相近相似的操作逻辑和操作应用数据、状态，以类的型式描述出来，以对象实例的形式在软件系统中复用，以达到提高软件开发效率的作用。

在Python中赋值都是引用赋值，当一个对象被创建之后会将其引用赋值给一个变量，此时这个对象的引用计数就为1。
每当这个对象的引用被保存至一个变量中时，该对象的引用计数就会加一，使用del可以删除某个引用，此时对象的引用计数会减一。
当一个对象的引用计数为0的时候，该对象就会被删除，删除前会调用del方法
del方法也相当于C++中的析构方法

类属性和实例属性

在函数中定义的属性就是实例属性，也可以称之为对象属性，一个对象创建出来的时候它之中的属性就是实例属性。
而类属性就是一个类的属性，是定义在函数之外的，它被所有类的对象所共有，在内存中只有一片空间，和C++中的静态成员变量有类似之处。
对于公有的类属性，在类外可以通过类和对象对其进行访问。

如果需要在类外修改类属性，必须通过类对象去引用然后进行修改。
如果通过实例对象去引用，会产生一个同名的实例属性，这种方式修改的是实例属性，不会影响到类属性，并且之后如果通过实例对象去引用该名称的属性，实例属性会强制屏蔽掉类属性，即引用的是实例属性，除非删除了该实例属性。

异常的传递

如果try嵌套，那么如果里面的try没有捕获到这个异常，那么外面的try会接收到这个异常，然后进行处理，如果外边的try依然没有捕获到，那么再进行传递。
如果一个异常是在一个函数中产生的，例如函数A—->函数B—->函数C,而异常是在函数C中产生的，那么如果函数C中没有对这个异常进行处理，那么这个异常会传递到函数B中，如果函数B有异常处理那么就会按照函数B的处理方式进行执行；如果函数B也没有异常处理，那么这个异常会继续传递，以此类推。如果所有的函数都没有处理，那么此时就会进行异常的默认处理，即通常见到的那样

模块

有过C语言编程经验的朋友都知道在C语言中如果要引用sqrt函数，必须用语句#include <math.h>引入math.h这个头文件，否则是无法正常进行调用的。
那么在Python中，如果要引用一些其他的函数，该怎么处理呢？
在Python中有一个概念叫做模块（module），这个和C语言中的头文件以及Java中的包很类似，比如在Python中要调用sqrt函数，必须用import关键字引入math这个模块，下面就来了解一下Python中的模块。
说的通俗点：模块就好比是工具包，要想使用这个工具包中的工具(就好比函数)，就需要导入这个模块

模块的导入

在Python中用关键字import来引入某个模块，比如要引用模块math，就可以在文件最开始的地方用import math来引入。
格式如下：import 模块名1, 模块名2, ...
当解释器遇到import语句时，如果模块在当前的搜索路径就会被导入。
在调用math模块中的函数时，必须这样调用：模块名.函数名
如：

有时候我们只需要用到模块中的某个函数，只需要引入该函数即可，此时可以用下面方法实现：
from 模块名 import 函数名1, 函数名2, ...
通过这样的方法，不仅可以引入函数，还可以引入一些全局变量、类等
注意：通过这种方式引入的时候，调用函数时只能给出函数名，不能给出模块名，但是当两个模块中含有相同名称函数的时候，后面一次引入会覆盖前一次引入。也就是说假如模块A中有函数function()，在模块B中也有函数function()，如果引入A中的function在先、B中的function在后，那么当调用function函数的时候，是去执行模块B中的function函数。
如果想通过from一次性引入某个模块中所有的东西，可以使用通配符，格式如下：
from 模块名 import *
from语句让你从模块中导入一个指定的部分到当前的命名空间中，from不会把整个模块导入到命名空间，只会将指定的部分导入到命名空间。通过from语句导入的模块，调用其函数时不需要通过模块名来调用。

某些具有相同作用的模块可以放进一个目录下，这个目录就是一个包，我们可以将这个包（也就是目录）直接导入，以获取目录下中模块的功能。

然后就通过msg.recvmsg.函数()来对模块中的内容进行调用

我们也可以通过from 目录 import *的方式导入目录下的所有模块，但是需要目录下有__init__.py文件，这个文件控制着包的导入行为。
如果这个文件为空，那就仅仅把包导入，不会导入其中的模块。
如果要导入包中的模块，还需要在__init__.py文件中加一个__all__变量，写上将哪些模块导入，这样通过from 目录 import *的方式导入包的时候才能将包下的模块成功导入。这个__all__变量的用法和模块中__all__变量的用法一样。
通过from 目录 import *的方式导入包中的模块之后，调用就使用模块名.函数名进行调用。
__init__.py中还可以写上其他的语句，这些语句也能在包被导入的时候被执行（事实上这个__init__.py也相当于一个模块），如果__init__.py中定义了某些函数的时候，调用可以通过包名.函数()来进行调用。
