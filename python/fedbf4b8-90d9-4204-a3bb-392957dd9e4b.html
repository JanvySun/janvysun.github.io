变量定义

Python 中，每个变量在使用前都必须赋值，变量赋值以后该变量才会被创建。

等号 = 用来给变量赋值，等号左边是一个变量名，等号右边是存储在变量中的值。

因此定义变量的方法：**变量名 = 值**

如果想要数据变量的内容，用 print(NAME) 即可打印出来变量的内容

如果在交互式界面，直接输入变量名回车即可打印变量内容

变量类型

在内存中创建一个变量，会包括：

  1. 变量的名称
  2. 变量保存的数据
  3. 变量存储数据的类型
  4. 变量的地址

注意：

  • 在 Python 中，定义变量时是不需要指定变量的类型的
  • Python 解释器运行的时候，会自动根据赋值语句等号右侧的数据来自动推导出变量中保存数据的准确类型
  • 变量类型分为数字型和非数字型
  • 其中数字型有: 整型 (int)、浮点型 (float)、布尔型 (bool)
  • 非数字型有: 字符串 (string)、列表、元组、字典等

如果想要查看某个变量的类型，在交互式界面下可以使用 type(NAME) 函数来查看

在 Python 中，数字型变量之间是可以直接计算的，如变量是 bool 类型的在计算时True 对应的值为 1，False 对应的值为 0

字符串的计算：字符串之间可以使用 “+“ 号拼接生成新的字符串，也可以使用 “*“ 号来重复拼接相同的字符串。

注意：数字型变量和字符串之间不能进行其他计算



变量的转换

Python 中的变量之间可以进行变量转换，需要用到类型转换函数。转换函数如下表所示：

函数说明
int(x)将 x 转换成整型变量
long(x)将 x 转换成长整型变量
float(x)将 x 转换成字符型变量
str(x)将 x 转换成字符串
repr(x)将对象 x 转换成表达式字符串
eval(str)计算字符串 x 中的有效表达式，返回计算后的结果
tuple(x)将序列 x 转换成元组
list(x)将序列 x 转换成列表
chr(x)将整数 x 转换成字符
ord(x)将字符 x 转换成整数值
hex(x)将整数 x 转换成十六进制字符串
oct(x)将整数 x 转换成八进制字符串



变量的命名

标识符可以由字母、下划线和数字组成，并且不能以数字开头，标识符不能与关键字重名

局部变量和全局变量

  • 局部变量：定义在函数内部的变量，不同的函数可以定义相同名字的局部变量，但是各用各的不会对其他函数中的变量产生影响。

  • 全局变量：定义在函数外边的变量，它既能在一个函数中使用，也能其他函数中使用，这就是全局变量。

当函数中的局部变量名字和全局变量名字相同时，该变量在函数中只是一个局部变量，不能对全局变量产生影响，

如果要在函数中对一个全局变量进行修改，则需要使用 global 关键字将该变量声明为全局变量，这样才能对其进行修改。
