HTML(Hyper Text Markup Language): 超文本标记语言

  • 超文本: 是用超链接的方法，将各种不同空间的文字信息组织在一起的网状文本.
  • 标记语言: 由标签构成的语言。

html文档后缀名有：.html或者.htm
标签分为两种：

  • 围堵标签：有开始标签和结束标签。如<html> </html>
  • 自闭和标签：开始标签和结束标签在一起。如<br/>

html的标签不区分大小写，但是建议使用小写。

HTML文档规范

HTML 目前有 2 种主流文档类型，一个是 XHTML 1.0，另一个是 HTML5。两种文档区别：文档声明和编码声明有区别，HTML5 新增了标签元素以及元素属性。

xhtml 制定了文档的编写规范，html5 可部分遵守，也可全部遵守，看开发要求。

  • 所有的标签必须小写
  • 所有的属性必须用双引号括起来
  • 所有标签必须闭合
  • img 必须要加 alt 属性(对图片的描述)

文件标签

文件标签是构成 html 最基本的标签

  • <!DOCTYPE html>: 声明该文档是html文档(html5中的声明)
  • html: html文档的根标签，其中可以指定语言，如lang="zh_CN"
  • head: 头标签，用于指定html文档的一些属性。引入外部的资源
  • title: 标题标签
  • body: 体标签，网页上显示的内容

文本标签

  • <h1>/<h2>/<h3>/<h4>/<h5>/<h6>: 标题标签，h1~h6 字体大小逐渐递减
  • <p>: 段落标签，一个段落含有默认的上下间距，段落之间会用这种默认间距隔开
  • <br/>: 换行标签，如果文件中有换行，在网页中是不会显示的，要想显示换行需要用到 <br> 标签。该标签是自闭和标签
  • <hr/>: 展示一条水平线，该标签是自闭和标签
  • <b>: 字体加粗
  • <i>: 字体斜体
  • <font>: 字体标签
  • <center>: 文本居中

常用属性

  • color: 颜色
    • 英文单词:red,green,blue
    • rgb(值1，值2，值3): 值的范围：0~255 如 rgb(0,0,255)
    • #值1值2值3: 值的范围：00~FF之间。如： #FF00FF
  • width: 宽度
    • 数值：width=’20’ ,数值的单位，默认是 px(像素)
    • 数值%：占比相对于父元素的比例
  • size: 高度(尺寸)
  • align: 对其方式
    • center: 居中
    • left: 左对齐
    • right: 右对齐
  • face：字体，如：face="楷体"

块标签

  • div: 块标签，表示一块内容，没有具体语义
  • span: 行内元素，表示一行中的一段内容，没有具体语义

注：块标签通常配合 CSS 来实现

列表标签

  • ol: 表示一个有序列表，其子标签中的内容会按顺序展示
    ol标签中可以使用type属性，如：type="A" start="5"表示用大写字母来构成列表，从第5个字母(E)开始
  • ul: 表示一个无序列表，子标签中的内容不会按顺序展示
    ul标签中也可以使用type属性，常用值为有：disc(实心圆点)、square(实心方点)、circle(空心圆点)

列表子标签：

  • li: 列表中的子标签

链接标签

a: 定义一个超链接

特有属性：

  • href：指定访问资源的URL(统一资源定位符)
    该属性可以写#号，表示跳转到当前页面的顶部;
    可以写#+其他标签id，表示跳转到该id部分(锚点)
  • target：指定打开资源的方式
    _self: 默认值，在当前页面打开
    _blank: 在空白页面打开

图片标签

图片标签由 img 标签来实现，可以在页面上插入一张图片

特有属性：

  • src：指定图片地址，可以使用相对路径和绝对路径或者URL
  • alt：定义图片加载失败时显示的文字

语义化标签

html5中为了提高程序的可读性，提供了一些标签。

如下：

  • <header>：页眉
  • <footer>：页脚

表格标签

在HTML中用table标签来包括一个表格。
其中可以用tr标签定义表格的一行，在tr标签中可以使用td或者th标签来定义一个单元格
其中td标签代表一个普通的单元格，th代表表头单元格。

表格中常用属性：

  • width：宽度(单元格中的宽度一般使用百分比)
  • height：高度
  • border：边框，默认为0也就是没有边框，设置为1表示有边框
  • cellpadding：定义单元格中内容和边框的距离
  • cellspacing：定义单元格之间的距离。如果指定为0，则单元格的线会合为一条。
  • bgcolor：背景色
  • align：设置单元格中的水平对齐方式，属性值有left/center/right
  • valign：设置单元格中内容的垂直对齐方式，属性值有top|middle|bottom
  • colspan：设置单元格水平合并(合并列)，值为合并的数量
  • rowspan：设置单元格垂直合并(合并行)，值为合并数量

表格中有几个常用标签：

  • caption: 表格标题
  • thead: 表示表格的头部分
  • tbody: 表示表格的体部分
  • tfoot: 表示表格的脚部分

表单

概念: 用于采集用户输入的数据的。用于和服务器进行交互。
form: 用于定义表单的。可以定义一个范围，范围代表采集用户数据的范围
常用属性：

  • action：指定提交数据的URL
  • method：指定提交方式
    分类：一共7种，2种比较常用
    get: 请求参数会在地址栏中显示。
    post: 请求参数不会再地址栏中显示。

注意：表单项中的数据要想被提交：必须指定其name属性

input 标签

input: 可以通过type属性值，改变元素展示的样式
常用的属性如下：
type：该属性用于设置输入格式，常用的有:

  • text：文本输入框，默认值
  • password：密码输入框(特殊的文本输入框)
    以上文本输入框可以加入placeholder属性，指定输入框的提示信息，当输入框的内容发生变化，会自动清空提示信息
  • radio：单选框
    注意：
    要想让多个单选框实现单选的效果，则多个单选框的name属性值必须一样。
    一般会给每一个单选框提供value属性，指定其被选中后提交的值
    checked属性，可以指定默认值
  • checkbox：复选框
    注意：
    多个复选框的name属性值必须一致，一般会给每一个单选框提供value属性，指定其被选中后提交的值
    checked属性，可以指定默认值
  • file：文件选择框
  • hidden：隐藏域，用于提交一些信息。
  • color：取色器
  • date：日期
  • datetime-local：日期时间
  • email：邮箱，提交表单的时候会检测邮箱格式是否正确
  • number：数字，只能填写数字
  • submit：提交按钮。可以提交表单
  • reset：重置按钮，用于重置所有的输入信息
  • button：普通按钮
  • image：图片提交按钮
    src属性指定图片的路径
  • label：指定输入项的文字描述信息
    注意：label的for属性一般会和input的id属性值对应。如果对应了，则点击label区域，会让input输入框获取焦点。

name：该属性用于指定在传给服务器时的属性名
value：该输入的默认值，提交的数据就是该属性值

select标签

select: 下拉列表标签
其中的子标签为option，就是定义下拉条中的内容，option中定义value属性就行了。

textarea标签

textarea：文本域
常用属性如下：

  • cols：指定列数，每一行有多少个字符
  • rows：默认多少行。

内嵌框架

iframe标签会创建包含另外一个html文件的内联框架(即行内框架)
其src属性来定义另一个html文件的引用地址，frameborder属性定义边框，scrolling属性定义是否有滚动条
用法如下：
<iframe src="http://www..." frameborder="0" scrolling="no"></iframe>
