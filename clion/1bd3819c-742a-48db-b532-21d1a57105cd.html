Clion 集成 QT 开发

发表于2022-05-19|更新于2022-05-19|Software
|阅读量:

安装

Clion 官网 下载安装程序，安装该过程不详细介绍，可以自定义安装路径。

Clion 的配置可参考 IDEA 的配置

注：本文使用的 Clion 版本为 2023.2，QT 版本为 5.15.2

配置 MinGW

Settings -> Build,Execution,Deployment -> Toolchains 中 添加 MinGW 工具链，其中 Enviroment 配置 QT 的 MinGW 路径，如下：

配置外部工具 QT Designer

Settings -> Tools -> External Tools，在其中添加一个外部工具，如下：

添加完成之后，可以通过 QtDesigner 打开 ui 文件。

创建 QT 项目

打开 Clion 创建新项目，选择 Qt Widgets Executable，在 QT CMake prefix path 中填入 QT MinGW 的路径即可创建项目。

注：由 Clion 创建的项目无法编译，需对 CMakeList 文件进行改造
