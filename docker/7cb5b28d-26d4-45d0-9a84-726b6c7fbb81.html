容器相关

创建并运行容器

  • --name NAME：指定容器名
  • -d：在后台启动容器
  • -p O_PORT:I_PORT：端口映射（映射端口号：容器内端口号）
  • -i：以交互模式运行容器，通常与-t一起使用
  • -t：分配一个伪终端 docker run -it IMAGE-NAME /bin/bash

查看容器日志

选项：

  • -t：加入时间戳
  • -f：跟随最新的日志打印
  • --tail NUM：显示最后多少条，跟随显示最新日志：docker logs --tail 0 -f 容器id

查看运行的容器

  • -a：正在运行的和历史运行过的容器
  • -q：静默模式，只显示容器id
