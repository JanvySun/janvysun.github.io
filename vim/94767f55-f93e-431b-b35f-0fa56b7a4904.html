Vim 使用

发表于 2021-03-30 更新于 2021-06-30 分类于 Software 阅读次数：
本文字数： 2k 阅读时长 ≈ 4 分钟

Vim 是一个类似于 Vi 的著名的功能强大、高度可定制的文本编辑器，在 Vi 的基础上改进和增加了很多特性。Vim普遍被推崇为类Vi编辑器中最好的一个，事实上真正的劲敌来自 Emacs 的不同变体。1999 年 Emacs 被选为 Linuxworld 文本编辑分类的优胜者，Vim 屈居第二。但在 2000 年 2 月 Vim 赢得了 Slashdot Beanie 的最佳开放源代码文本编辑器大奖，又将 Emacs 推至二线， 总的来看，Vim 和 Emacs 在文本编辑方面都是非常优秀的。本文列出 vim 的一些操作。

打开文件

Vim 一共三大模式：编辑模式(命令模式)、输入模式、末行模式。打开文件默认处于编辑模式。

  • vim +#：打开文件，并定位于第#行
  • vim +：打开文件，定位至最后一行
  • vim +/PATTERN：打开文件，定位至第一次被 PATTERN 匹配到的行的行首

打开多个文件

vim FILE1 FILE2...（默认显示第一个文件）

末行模式下输入 next 则进入下一个文件，prev 进入前一个文件，last 表示切换至最后一个文件，first 表示切换至第一个文件

分屏打开多个文件

  • vim -o FILE1 FILE2：水平显示2个窗口
  • vim -O FILE1 FILE2：垂直显示2个窗口

关闭文件

  • :q：退出
  • :wq：保存并退出
  • :q!：不保存并退出
  • :x：相当于 wq
  • ZZ:：保存并退出

编辑模式命令

移动光标

  • h: 光标左移
  • j: 光标下移
  • k: 光标上移
  • l: 光标右移
  • w: 移至下一个单词的词首
  • e: 跳至当前或下一个单词的词尾
  • b: 跳至当前或前一个单词的词首

行内跳转

  • 0: 绝对行首
  • ^: 行首的第一个非空白字符
  • $: 绝对行尾

行间跳转

  • gg：跳转至行首
  • NUMG：跳转至第NUM行；
  • G：最后一行

末行模式下，直接给出行号即可完成行间跳转

翻屏

  • Ctrl+f: 向下翻一屏
  • Ctrl+b: 向上翻一屏
  • Ctrl+d: 向下翻半屏
  • Ctrl+u: 向上翻半屏

进入输入模式

  • i: 在当前光标所在字符的前面，转为输入模式
  • a: 在当前光标所在字符的后面，转为输入模式
  • o: 在当前光标所在行的下方，新建一行，并转为输入模式
  • O：在当前光标所在行的上方，新建一行，并转为输入模式
  • I：在当前光标所在行的行首，转换为输入模式，相当于 ^i
  • A：在当前光标所在行的行尾，转换为输入模式，相当于 $a
  • ce/cw：将光标所在位置到单词尾的字符删除，并进入输入模式，相当于 dei
  • ciw：将光标所在位置的单词删除，并进入输入模式，相当于 diwi
  • C：将光标所在位置一直到行尾的字符删除，并进入输入模式，相当于 Da
  • s：将光标所在位置的字符删除，并进入输入模式，相当于 xi
  • S：将光标所在位置的行删除（保留缩进），并进入输入模式，相当于 ^C

注：如果要从输入模式回到编辑模式，则需要按 ESC 键

删除

  • x：删除光标所在处的单个字符
  • NUMx：删除光标所在处及向后的共NUM个字符
  • dw/de/db：根据单词删除光标前后内容
  • dd：删除当前光标所在行
  • D：删除光标到行尾的内容

复制

  • y：用法同 d 命令

粘贴

  • p：如果删除或复制为整行内容，则粘贴至光标所在行的下方，如果复制或删除的内容为非整行，则粘贴至光标所在字符的后面；
  • P：如果删除或复制为整行内容，则粘贴至光标所在行的上方，如果复制或删除的内容为非整行，则粘贴至光标所在字符的前面；

替换

  • r：替换单个字符
  • R：进入替换模式

撤消编辑操作

  • u：撤消前一次的编辑操作
  • Ctrl + r：撤消最近一次撤消操作

重复前一次编辑操作

按 .（点）可重复前一次编辑操作

拆分窗口

  • Ctrl-w s：水平拆分窗口
  • Ctrl-w v：垂直拆分窗口

末行模式

删除

StartADD,EndADDd （如1,3d表示删除1到3行）

  • .：表示当前行
  • $：最后一行
  • +NUM：向下的NUM行

删除空白行

  • g//：全区命令
  • ^\s*$：匹配空行
  • d：删除

查找

  • /PATTERN
  • ?PATTERN

查找并替换

在末行模式下使用 s 命令（用法和 sed 中的 s 相同）

分组替换

如将 <h1>Hello</h1> 替换为 <span>Hello</span>

分屏

  • :vsp FILENAME：垂直分屏打开某个文件
  • :sp FILENAME：水平分屏打开某个文件

在窗口间切换光标：Ctrl-w，上下左右光标

截取文件内容

  • :[ADDR1,ADDR2]w [/path/to/somewhere]：将文件中的ADDR1行至ADDR2行保存在另一个文件中

填充文件

  • :r /path/to/somewhere：将另一个文件的所有内容填充至当前光标后

可视化模式

逐行

  • v：按字符选取
  • V：按行选取
  • viw：选取当前光标的单词
  • vis/vip：选取当前光标所在段落
  • gu/gU：选中之后，gu 或者 gU 改变大小写

逐列

  • Ctrl-v：进入列选择
