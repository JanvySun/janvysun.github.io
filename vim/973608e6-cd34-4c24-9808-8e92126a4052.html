Vim 插入及可视模式

发表于2023-03-22|更新于2023-03-22|Software
|字数总计:753|阅读时长:3分钟|阅读量:

计算器

:h i_ctrl-r 查看帮助
=7*7 显示结果
=10/2 显示结果
在插入模式下，输入ctrl-r{register}就可以加入寄存中的内容；
在插入模式下，按 ctrl+r ,然后再按= 就可以进行表达式寄存器，在这时就可以输入加减乘除，最后的结果输入的文本当中；

当一些特殊的字符在键盘中不容易输入时，这时可以通过输入相应字符的编码来输入它们；
方法为： ctrl + v ｛code｝
VIm中的编码共3位数字，如A的编码为65，这时，就可以输入： ctrl+v,再输入065就可以；
不过对于使用unicode编码的字符，用的为4位16进行进行编码的，此时的输入为：ctrl+v，再输入u， 然后再输入编码；
如：¿ 为00bf, 此时，输入 ctrl-v, u, 00bf。
使用 ga 命令可以把光标上字符的编码显示出来；
使用：h i_ctrl-v_digit 看更多详细内容；

ctl+p：自动补全
ctl+n：自动补全
ctrl-h：删除前一个字符;
ctrl-w：删除前一个单词;
ctrl-u：删除至行首;

vi[ 选择一对 [] 中的所有字符
vi( 选择一对 () 中的所有字符
vi< 选择一对 <> 中的所有字符
vi{ 选择一对 {} 中的所有字符
vit 选择一对 HTML/XML 的标签内部的所有字符
vi” vi’ vi选择一对引号字符 (” 或 ‘ 或) 中所有字符
aw a word (with white space) |v_aw|
iw inner word |v_iw|
aW a WORD (with white space) |v_aW|
iW inner WORD |v_iW|
as a sentence (with white space) |v_as|
is inner sentence |v_is|
ap a paragraph (with white space) |v_ap|
ip inner paragraph |v_ip|
ab a () block (with parentheses) |v_ab|
ib inner () block |v_ib|
aB a {} block (with braces) |v_aB|
iB inner {} block |v_iB|
at ablock (with tags) |v_at|
it innerblock |v_it|
a< a <> block (with <>) |v_a<|
i< inner <> block |v_i<|
a[ a [] block (with []) |v_a[|
i[ inner [] block |v_i[|
a” a double quoted string (with quotes) |v_aquote|
i” inner double quoted string |v_iquote|
a’ a single quoted string (with quotes) |v_a’|
i’ inner simple quoted string |v_i’|
aa string in backticks (with backticks) |v_a|
iinner string in backticks |v_i|

i / a 在光标前/后插入
I / A 插入行首/行尾
o / O （字母） 在下方/上方插入新行
s / S 删除字符/行并插入
C 更改为行尾
gi 在最后一个插入点插入
Esc | <C-[> 退出插入模式

visual 模式

d | x 删除选择 （剪切）
s 替换选择
y Yank 选择（复制）

v 进入视觉模式
V 进入视线模式
进入视觉块模式
ggVG 选择所有文本

/ < 向右/向左移动文本

文本对象

​va”​ 选择引号 “…”
​vi[​ 选择内括号 [ …]
​va[​ 选择括号 […]
viw 选择内词
vip 选择内部段落
vipip 选择更多段落

vU 大写字符
vu 小写字符
viw U 大写单词
viw u 小写单词

Vu / guu 小写行
